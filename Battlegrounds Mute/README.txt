******TUTORIAL********

> Install Autohotkey ( https://autohotkey.com/ )
> Right click mute.ahk
> Edit with your favourite Text Editor/IDE
> Replace my file location with yours.
> Save
> Put mute.ahk in startup folder
> Mute.ahk will run whenever your computer turns on but will do nothing unless you press Page up
